# ⏱️ Time Management System (Java)

A simple Java-based Time Management program that helps users organize tasks, track time, and improve productivity. This project demonstrates core Java concepts like OOP, user input handling, and basic logic building.

---

## 🚀 Features

- 📋 Add and manage daily tasks  
- ⏰ Track time spent on tasks  
- ✅ Mark tasks as completed  
- 📊 View task summary  
- 🧠 Improves productivity and time discipline  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **Concepts Used:**
  - Classes & Objects  
  - Methods  
  - Arrays / ArrayList  
  - Loops & Conditionals  
  - User Input (Scanner)  

---


---

## ⚙️ How It Works

1. User runs the program  
2. Menu options are displayed:
   - Add Task  
   - View Tasks  
   - Complete Task  
   - Exit  
3. User interacts via console input  
4. Tasks are stored and managed during runtime
